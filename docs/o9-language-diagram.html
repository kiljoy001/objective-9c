o9 Language — How It Works

Every object is a CSP actor. The application is one 9P fileserver. Method calls are channel messages locally and 9P transactions remotely. The compiler always emits Plan 9 C.

Types & Declarations

KeywordDescriptionExample
class Define a class. Each new spawns a CSP actor (proc + dispatch_chan). The application is one shared 9P server; classes register handlers, not separate fileservers. class Counter { int64 val; method Counter(int64 n) { val = n; } method int64 get() { return val; } }
interface Compile-time contract. Cannot be instantiated. interface Sink { method void put(int64 value); }
struct Value type — no actor, no 9P server, embedded by copy. struct Point { int64 x; int64 y; }
enum Named integer constants. Compile-time only. enum Color { Red, Green, Blue }
module Namespace grouping. Classes qualified as App.Counter. module App { class Counter { ... } }
object Singleton instance declaration. Class-typed slot. object Logger { method void log(string s) { ... } }
import Splice another .o9 file's declarations into the source. import "string.o9";
main Reserved program entry block. It is not a user class method and does not use legacy func main() syntax. main { print("hello\n"); }
function Spawnable function-class. Desugars to anonymous one-method class. function rawadd(int64 a, int64 b) int64 { return a + b; }
method Class member function. Dispatched via CSP channel or 9P. method int64 add(int64 n) { val = val + n; }
~ClassName Destructor. Runs synchronously on delete, then actor exits. ~Res() { print("destructor ran\n"); }

Field Modifiers

KeywordDescriptionExample
prop Property field. Exposed as a 9P file. Read/write through the facade. prop byte value;
state Authoritative state field. Persisted to state .tab file. state int64 counter;
atomic REMOVED. Using it is a compile error. Atomics are internal runtime machinery or raw C inside a function. atomic int64 n; // ERROR: 'atomic' is not a language keyword
stream Typed CSP channel field. Auto-created at construction. Buffered (8). Carries o9 values through the generic channel envelope. stream s; method void put(int64 v) { s -> v; }
chan Alternate spelling of stream. Same NStream node. chan c; method int64 take() { return <- c; }
secret Sealed storage. Desugars to blob + seal/open methods. AEAD. secret string token; // desugars to: // string token__blob; // seal_token(key, v) → encrypt // open_token(key) → decrypt
cap REMOVED. Using it is a compile error. Capabilities are 9P fids / namespace mounts, not a field type. Use a secret field for bearer tokens. // cap string auth; ← ERROR: 'cap' is not a language keyword // Use namespace mounts for authority, // or a secret field for a bearer token
private Class-scoped member. Not reachable through the 9P facade. private int64 val; private method void bump(int64 d) { ... }
public Default. Reachable from outside and through 9P facade. public method int64 get() { return val; }
abstract Marks a class non-instantiable or a method bodiless. abstract class Base { abstract method int64 id(); }

Control Flow & Error Handling

KeywordDescriptionExample
if / else / elif Conditional branches. elif = else if. if(n > balance) { fail("insufficient"); } elif(n > 0) { balance -= n; } else { fail("zero"); }
while Loop with condition. No do-while, no break/continue. while(n == 1) { print(t.get("item"), "\n"); n = t.next(); }
for C-style loop: init; cond; step. Semicolon-separated. for(i = 0; i < 5; i = i + 1) { sum = sum + i; }
return Return a value from a method/function. return val * 2;
try Propagate callee's error. Errors are values (fail sets err). int64 x = try leaf(n); // if leaf() called fail(), x = 0, error propagates
defer Run cleanup at method exit, on every path (success or error). defer close(h); // runs when method returns, regardless of try/fail
fail Set error string and return early. Builtin, not a keyword. if(n > balance) { fail("insufficient funds"); }
delete Run destructor synchronously, then actor exits. Blocks. Res r = new Res(5); delete r; print("after delete\n");
new Construct object: spawn actor proc, create dispatch_chan, init fields. Not a 9P server — the app's shared server routes to it. Counter c = new Counter(10);
near Read exports/name.tab over IL. Data only; remote objects are rejected. sync/push refresh or upload Tabula text. near Tabula t = new Tabula("orders", "item,qty") @ "il!host!9999";
far Read exports/name.tab over TCP. Data only; remote objects are rejected. far Tabula t = new Tabula("orders", "item,qty") @ "tcp!host!9999";
listener Serve local Tabula exports and inert inbound imports through the app's 9P tree. listener Tabula t = new Tabula("orders", "item,qty") @ "il!*!9999";
spawn Run a function-class concurrently. Returns Task<T>. Functions may be top-level or nested in a class. Task<int64> t = spawn rawadd(10, 20); int64 v = t.await();
cast<T> Explicit scalar conversion. Integers, chars, byte, and bool are allowed. Object, string, collection, and pointer casts are rejected. byte b = cast<byte>(260); // 4 bool ok = cast<bool>(b); // 1 int64 n = cast<int64>(ok); // 1

Built-in Types & Collections

KeywordDescriptionExample
int64..int8 Plan 9 scalar storage types. Map to vlong/uvlong, long/ulong, short/ushort, char/uchar. int64 n; uint8 b; int16 s;
string O9String* — refcounted, UTF-8. Builtins: len, cmp, cat. string s = "hello"; print(len(s), "\n");
bool Boolean. true/false literals. Stored as int. bool ok = true;
byte Unsigned 8-bit. Same as uint8/uchar. prop byte value;
double Plan 9 C double. Used by raw-C function wrappers and Math. Prints with %g. double x = 9.0; print(x, "\n");
intptr / uintptr Address-sized integer values for raw-C function interop. Rejected on object fields, method params, and method returns. function fdaddr(int64 fd) uintptr { c { /* return opaque address-sized value */ } }
void No return value. method void set(int64 n) { val = n; }
List<T> Compiler collection carrier. O9Slice backing. Prefer stdlib list<T> for object-style methods. List<int64> numbers; numbers.Add(10); print(numbers[0]); // 10
Dict<K,V> Compiler hash-map carrier. O9Dict backing. Prefer stdlib dictionary<T> for object-style methods. Dict<string, int64> ages; ages.Set("bob", 42);
T[] Array type. O9Slice backing (same as List). Growable. int64[] items; items[0] = 42;
Task<T> Spawn join handle. await() blocks for result. Task<int64> t = spawn rawadd(3, 4); int64 v = t.await(); // 7
(T,U) Tuple return value. Lowered to an immutable generated C struct pointer; object handles are rejected as tuple fields. method (int64, int64) pair(int64 x) { return (x, x + 1); } int64 a; int64 b; (a, b) = obj.pair(7);
Tabula First-class table type. libtab-backed. Text boundary. Binary payloads use a 0x column with hex text. Tabula t = new Tabula("orders", "item,qty"); t.add("a"); t.set("item", "widget"); print(t.get("item")); // widget
MountTable Lower-level namespace composition as data. Prefer Namespace for normal code. MountTable mt = new MountTable(); mt.allowRoot("/tmp/o9"); mt.bind("/tmp", "tmp", 0); mt.dir("cache", 493); mt.validate(); mt.apply();
Namespace Object API over MountTable for programmatic namespace setup. Namespace ns = new Namespace(); ns.root("/tmp/o9"); ns.bindBeforeCreate("/tmp", "tmp"); ns.apply();
chan CSP channel field. Prefer typed chan<T> or stream<T>. Supports primitives, strings, structs, object handles, arrays, and List<T>; Dict payloads are deferred. chan<int64> c; stream<string> events;

Channel Operators & Raw C

OperatorDescriptionExample
-> Channel send. Copies the typed value into a generic CSP envelope and sends it. c -> v; // send v on channel c
<- Channel receive. Receives the envelope and copies the typed value into the target. x = <- c; // receive from c into x
->? Non-blocking send (try). Alt with CHANSND + CHANNOBLK. c ->? v; // try-send, no rendezvous
alt Select over receive-ready channels. Optional default maps to CHANNOBLK. alt { case x = <- leftc: print(x, "\n"); case x = <- rightc: print(x, "\n"); default: print("none\n"); }
c { } Raw Plan 9 C block. Inside function bodies only. class Worker { function rawadd(int64 a) int64 { int64 out; c { out = a + 7; } return out; } }
use { } Declare constrained C dependencies. Resolves via deps.tab. function countbytes(string path) int64 { use { bio } c { Biobuf *b = Bopen(path, OREAD); } }

Compilation

  • • yacc/lex grammar → o9c transpiler
  • • 4 phases: import, prescan, parse, typecheck
  • • Monomorphization for generics (Box<int64>)
  • • Emits Plan 9 C → 6c → 6l → binary
  • • Links libo9.a (runtime + dispatch)
  • • Target: 9front (6c/6l toolchain)

Object Model

  • • new = proccreate + dispatch_chan (CSP actor)
  • • Application is ONE 9P server with clone/session reply state
  • • Classes register handlers into O9ClassH[] table
  • • Method call = sendp(O9Msg) → recvp(O9Reply*)
  • • ARC ledger tracks references (ainc/adec)
  • • delete = synchronous destroy + threadexits
  • • Runtime atomics are internal only: ARC, tasks, counters

9P Facade

  • • ONE server per app: clone, session ctl/data/status, global methods, exports, imports
  • • Session ctl receives method lines: method Class.inst name arg0=5
  • • Session data returns the result as text
  • • Session pool uses explicit request state, not a global current session
  • • srvrelease/srvacquire for concurrency
  • • Remote: raw 9P (Tversion→Tattach→Twalk→Topen→Twrite/Tread)

Tabula, Namespace & MountTable

  • • Tabula = first-class table type (libtab backing)
  • • Every value in/out is string (text boundary)
  • • Namespace = normal object API for namespace setup
  • • MountTable = lower-level namespace composition as data
  • • Schema: call/fd/old/new/flag/aname/mode
  • • Validate-then-apply pattern (path traversal safe)
  • • Operations: root, dir, bind, mountsrv, validate, apply

Security

  • • secret string desugars to AEAD blob + seal/open
  • • Plaintext never exists as stored state
  • • Key custody stays with caller (passkey/exchange/keygen)
  • • Crypto: monocypher (Ed25519, X25519, BLAKE2b, AEAD)
  • • private members not reachable through 9P
  • • use { bio } constrains raw C dependencies
  • • Address-sized values stay out of object method surfaces
  • • Capabilities = 9P fids / namespace mounts, not a keyword

Concurrency

  • • One model: every object is a CSP actor
  • • function = anonymous one-method class
  • • spawn f(args) → Task<T> join handle
  • • t.await() blocks on done channel (per-proc error)
  • • alt selects over channel receives with optional default
  • • Channels buffered (8), boxes freed on recv or drain
  • • No goroutines, no closures, no capture

Standard Library

All stdlib modules are written in o9 itself — raw C operations wrapped in function bodies, exposed as class methods. Every operation that can block runs via spawn + Task<T>. E2e tested with expected output on 9front.

Stdlib Architecture

  • • Written in o9 itself — no separate C library
  • • Raw C in function bodies wraps Plan 9 libc calls
  • • Classes expose methods; blocking ops use spawn + await
  • • import "string.o9" splices declarations into source
  • • 13 modules, all e2e tested with expected output
  • • Constrained deps: use { bio } for buffered I/O, use { memdraw }/use { draw } for drawing
